Cocktail Cove - E-commerce Landing Page

Projeto de uma landing page de e-commerce para uma loja de coquetéis, com foco em experiência do usuário, design moderno e funcionalidades interativas como carrinho de compras e formulário de pedidos.
Acesse o projeto online: (https://69bc4a74d15d183818ac49e4--adorable-chimera-bb1d51.netlify.app/)

Objetivo do Projeto

Simular uma aplicação real de e-commerce, praticando conceitos essenciais do desenvolvimento front-end como:

Estruturação de layout moderno;
Responsividade (mobile-first);
Interatividade com JavaScript;
Experiência do usuário (UI/UX).

Tecnologias Utilizadas

HTML5;
CSS3;
JavaScript.

Funcionalidades

Listagem de produtos (coquetéis);
Favoritar itens;
Carrinho de compras com contador;
Layout totalmente responsivo;
Formulário de pedido;
Avaliações dos produtos.
